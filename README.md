# FocusFlow AI

## Problema que resuelve
Muchas personas tienen objetivos importantes pero se bloquean porque no saben por dónde empezar, cuánto tiempo les llevará o cómo organizarse sin agotarse.

## Para quién es
Personas con poco tiempo y mucha carga mental que necesitan convertir una meta en pasos pequeños y un plan diario realista.

## Por qué existe
Porque la mayoría de las herramientas de productividad son genéricas. FocusFlow AI usa inteligencia artificial para transformar una intención en un plan concreto, ejecutable y humano.

## Qué hace (MVP)
- El usuario escribe una tarea u objetivo
- La app genera pasos pequeños, ordenados y con estimación de tiempo
- El usuario puede guardar el plan y marcar progreso

## Tech stack (provisional)
- Frontend: React + TypeScript
- Backend: Node.js + Express
- Base de datos: por definir
- IA: generación de pasos mediante prompting
